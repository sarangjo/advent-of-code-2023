- https://adventofcode.com/
- https://raygun.com/blog/programming-languages/

## How to run

1. Golang
    ```
    $ go run day1.go
    ```
2. C++17 (with GCC)
    ```
    $ g++ -std=c++17 -o day2[.exe] day2.cpp
    $ ./day2
    ```
3. Python
    ```
    $ python day3.py
    ```
4. Rust
    ```
    $ rustc day4.rs
    $ ./day4
    ```
5. Typescript (with NodeJS)
    ```
    $ npm install
    $ npx tsc
    $ node ./out/day5.js
    ```
6. Java
    ```
    $ javac day6.java
    $ java day6
    ```
7. C# (with .NET)
    ```
    $ dotnet run
    ```
8. Swift
    ```
    $ swiftc day8.swift
    $ ./day8
    ```
9. PHP
    ```
    $ php day9.php
    ```
10. Ruby
    ```
    $ ruby day10.rb
    ```
11. Perl
    ```
    $ perl day11.pl
    ```
