# Advent of Code 2023

https://adventofcode.com/

## How to run

1. Golang
    ```
    $ go build day1.go
    $ ./day1.exe
    ```
2. C++17
    ```
    $ g++ -o day2.exe day2.cpp
    $ ./day2.exe
    ```
3. Python
    ```
    $ python day3.py
    ```
4. Rust
    ```
    $ rustc day4.rs
    $ ./day4.exe
    ```
5. Typescript (with NodeJS)
    ```
    $ npm install
    $ npx tsc day5.ts
    $ node day5.js
    ```
